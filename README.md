# K8s Foundation — Kubernetes Learning

Lộ trình học Kubernetes từ cơ bản → nâng cao, tập trung thực hành deploy và vận hành hệ thống.

---

## 📚 Nội dung

1. **Architecture** — Kiến trúc K8s, Control Plane, Node
2. **Cluster Setup** — Cài cluster (kubeadm, containerd, CNI)
3. **Workloads** — Pod, Deployment, StatefulSet, Job
4. **Networking** — Service, DNS, giao tiếp Pod
5. **Storage** — PV, PVC, StorageClass
6. **Ingress** — Expose HTTP/HTTPS
7. **Lifecycle** — Update, rollback, health check
8. **Scheduling** — Resource, affinity, taints
9. **Security** — RBAC, Secrets, NetworkPolicy
10. **Observability** — Logging, monitoring, debug
11. **Autoscaling** — HPA, VPA
12. **Backup & DR** — Backup, restore
13. **(Optional)** Multi-master cluster
14. **(Optional)** Longhorn storage

---

## 🎯 Mục tiêu

* Hiểu & deploy Kubernetes
* Chạy ứng dụng thực tế
* Debug & vận hành hệ thống
